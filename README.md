# AHA CRM

CRM minimalista offline con pipeline Kanban.

## Perfil: Lite

App offline-first para gestion de clientes y ventas.

## Modulos

- **Contactos** — CRUD con busqueda e historial de interacciones
- **Pipeline** — Kanban 5 etapas con drag & drop
- **Cotizaciones** — Creacion desde deals con PDF
- **Facturacion** — Generacion desde deal cerrado con folio automatico
- **Reportes** — Graficos de conversion funnel y export CSV

## Tecnologia

- Alpine.js + Dexie.js (IndexedDB)
- DaisyUI 4 + Tailwind CSS
- CryptoJS (cifrado offline)
- Chart.js + jsPDF
